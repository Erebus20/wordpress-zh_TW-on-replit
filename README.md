## 使用说明
- replit环境选择`PHP Web Server`

```bash
bash <(curl -s https://raw.githubusercontent.com/sxbai/wordpress-zh_CN-on-replit/main/build.sh)
```
- 执行上面脚本
- 最后点击Run运行项目即可。
