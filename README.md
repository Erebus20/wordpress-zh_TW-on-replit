## 使用說明
- replit環境選擇`PHP Web Server`

```bash
bash <(curl -s https://raw.githubusercontent.com/Erebus20/wordpress-zh_TW-on-replit/main/build.sh)
```
- 執行上面腳本
- 最後點擊Run運行項目即可。
